Features

⛓

On-Chain World State

Every block placement and destruction is recorded as a transaction on the VoxelChain EVM network with Merkle tree verification.

🎮

Browser-Based 3D

Play directly in your browser with Three.js WebGL rendering. No downloads, no installs. Desktop and mobile supported.

💰

Game Economy

Mine rare ores for VXL rewards. Trade blocks on the marketplace. Craft tools and materials with 12+ recipes.

👥

Multiplayer

Real-time player sync with avatar rendering. Chat with other players. See who's building what in the shared world.

🤖

AI Agent API

30+ programmatic commands for AI agents. A* pathfinding, structured state snapshots, and external process communication.

🔍

Block Explorer

Full-featured explorer with world overview, economy stats, player tracking, marketplace, crafting recipes, and faucet.
